# Đảo Ngược Chuỗi

## Mô tả bài toán
Viết chương trình đảo ngược một chuỗi sử dụng StringBuilder.

## Yêu cầu
1. Nhập một chuỗi
2. Đảo ngược chuỗi
3. In ra chuỗi đã đảo ngược

## Định dạng Input
- Một chuỗi (có thể chứa dấu cách)

## Định dạng Output
- Chuỗi đã đảo ngược

## Ví dụ

**Input:**
```
Hello World
```

**Output:**
```
dlroW olleH
```

**Input:**
```
Java Programming
```

**Output:**
```
gnimmargorP avaJ
```

**Input:**
```
12345
```

**Output:**
```
54321
```

## Gợi ý
- Sử dụng `StringBuilder` và phương thức `reverse()`
- Hoặc duyệt chuỗi từ cuối về đầu
- Nhớ dùng `toString()` để chuyển StringBuilder về String
