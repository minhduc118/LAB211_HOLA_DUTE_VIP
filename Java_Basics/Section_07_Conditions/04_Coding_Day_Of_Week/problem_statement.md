# Ngày Trong Tuần

## Mô tả bài toán
Viết chương trình nhập một số từ 1-7 và in ra tên ngày trong tuần tương ứng.

## Yêu cầu
1. Nhập một số nguyên từ 1-7
2. In ra tên ngày tương ứng:
   - **1**: Sunday
   - **2**: Monday
   - **3**: Tuesday
   - **4**: Wednesday
   - **5**: Thursday
   - **6**: Friday
   - **7**: Saturday
3. Nếu số không hợp lệ, in ra "Invalid"

## Định dạng Input
- Một số nguyên n

## Định dạng Output
- Tên ngày trong tuần hoặc "Invalid"

## Ví dụ

**Input:**
```
1
```

**Output:**
```
Sunday
```

**Input:**
```
5
```

**Output:**
```
Thursday
```

**Input:**
```
10
```

**Output:**
```
Invalid
```

## Gợi ý
- Có thể sử dụng `switch-case` cho bài này
- Đừng quên trường hợp `default` cho số không hợp lệ
