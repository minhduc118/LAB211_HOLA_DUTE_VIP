# Khai Báo Biến Cơ Bản

## Mô tả bài toán
Cho trước một số nguyên `n` và một chuỗi `name`. Hãy khai báo các biến phù hợp để lưu trữ và in ra màn hình theo format yêu cầu.

## Yêu cầu
1. Nhập một số nguyên `n`
2. Nhập một chuỗi `name`
3. In ra: `Name: [name], Age: [n]`

## Định dạng Input
- Dòng 1: Một số nguyên `n` (1 ≤ n ≤ 100)
- Dòng 2: Một chuỗi `name` (không chứa khoảng trắng)

## Định dạng Output
- In ra một dòng theo format: `Name: [name], Age: [n]`

## Ví dụ

**Input:**
```
25
John
```

**Output:**
```
Name: John, Age: 25
```

**Input:**
```
30
Alice
```

**Output:**
```
Name: Alice, Age: 30
```

## Gợi ý
- Sử dụng `Scanner` để nhập dữ liệu
- Khai báo biến `int` cho tuổi
- Khai báo biến `String` cho tên
- Sử dụng `System.out.println()` để in kết quả
