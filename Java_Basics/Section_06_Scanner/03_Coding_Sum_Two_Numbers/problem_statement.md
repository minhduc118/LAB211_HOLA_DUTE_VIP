# Tính Tổng Hai Số

## Mô tả bài toán
Viết chương trình nhập vào 2 số nguyên và in ra tổng của chúng.

## Yêu cầu
1. Nhập 2 số nguyên a và b
2. Tính tổng a + b
3. In ra kết quả

## Định dạng Input
- Dòng 1: Số nguyên a
- Dòng 2: Số nguyên b

## Định dạng Output
- In ra tổng của a và b

## Ví dụ

**Input:**
```
5
3
```

**Output:**
```
8
```

**Input:**
```
100
200
```

**Output:**
```
300
```

## Gợi ý
- Sử dụng `Scanner` để nhập dữ liệu
- Sử dụng `nextInt()` để đọc số nguyên
- Sử dụng toán tử `+` để tính tổng
