# Chuyển Đổi Kiểu Dữ Liệu

## Mô tả bài toán
Cho một số thực (double), hãy chuyển đổi và in ra số nguyên (int) tương ứng.

## Yêu cầu
1. Nhập một số thực
2. Chuyển đổi sang số nguyên (ép kiểu)
3. In ra số nguyên sau khi chuyển đổi

## Định dạng Input
- Một số thực (double)

## Định dạng Output
- Một số nguyên (int)

## Ví dụ

**Input:**
```
9.78
```

**Output:**
```
9
```

**Input:**
```
15.99
```

**Output:**
```
15
```

## Gợi ý
- Sử dụng `scanner.nextDouble()` để nhập số thực
- Sử dụng `(int)` để ép kiểu từ double sang int
- Phần thập phân sẽ bị cắt bỏ (không làm tròn)
