## Class Design

### Example:
- Create an **abstract class Shape** contains three methods `printResult`, `getPerimeter` and `getArea`.
- Create classes **Triangle**, **Rectangle**, **Circle** that extend from class Shape.
- Construct the shapes that consists of the properties of a circle (radius), a rectangle (width, length), a triangle (sideA, sideB, sideC) and generate their getter and setter methods.
- **Override** the methods of the Shape class.

---

## Formulas

### Rectangle:
- **Area** = width × length
- **Perimeter** = 2 × (width + length)

### Circle:
- **Area** = π × radius²
- **Perimeter** = 2 × π × radius
- Use `Math.PI` for π

### Triangle (Heron's formula):
- **Perimeter** = sideA + sideB + sideC
- **p** = Perimeter / 2 (semi-perimeter)
- **Area** = √(p × (p - a) × (p - b) × (p - c))
- Use `Math.sqrt()` for square root
