## Title
Letter and character count.

---

## Background Context

Knowing the number of words or characters in a document can be important. For example, if the author is required to write a minimum or maximum amount of words for an article or paper, word counter can help them know if their article meets these requirements.

---

## Program Specifications

Design a program that allows users to input the string value. Count and display the number of letters and characters.

---

## Expectation of User Interface

```
Enter your content:
hello world
{hello=1, world=1}
{w=1, d=1, e=1, r=1, o=2, l=3, h=1}
```

---

## Guidelines

To tokenize a string, you can use `StringTokenizer` in `java.util` package.

---

## Hướng dẫn chi tiết

### Yêu cầu bài toán:
1. Nhập một chuỗi văn bản
2. Đếm số lần xuất hiện của mỗi **từ** (word count)
3. Đếm số lần xuất hiện của mỗi **ký tự** (character count)
4. Hiển thị kết quả dưới dạng `{key=value}`

### Cách tiếp cận:

#### Đếm từ (Word Count):
```java
import java.util.StringTokenizer;
import java.util.HashMap;
import java.util.Map;

String input = "hello world";
StringTokenizer st = new StringTokenizer(input);
Map<String, Integer> wordCount = new HashMap<>();

while (st.hasMoreTokens()) {
    String word = st.nextToken();
    wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
}
System.out.println(wordCount);
```

#### Đếm ký tự (Character Count):
```java
Map<Character, Integer> charCount = new HashMap<>();

for (char c : input.toCharArray()) {
    if (c != ' ') {  // Bỏ qua khoảng trắng
        charCount.put(c, charCount.getOrDefault(c, 0) + 1);
    }
}
System.out.println(charCount);
```

### Input:
```
hello world
```

### Output:

**Word Count:**
```

```
- `hello` xuất hiện 1 lần
- `world` xuất hiện 1 lần

**Character Count:**
```
{w=1, d=1, e=1, r=1, o=2, l=3, h=1}
```
- `w` xuất hiện 1 lần
- `d` xuất hiện 1 lần
- `e` xuất hiện 1 lần
- `r` xuất hiện 1 lần
- `o` xuất hiện 2 lần (trong "hello" và "world")
- `l` xuất hiện 3 lần (trong "hello")
- `h` xuất hiện 1 lần

## Lưu ý
- Không đếm khoảng trắng trong character count
- Có thể dùng `HashMap` hoặc `LinkedHashMap` để lưu kết quả
- `StringTokenizer` mặc định tách theo khoảng trắng
